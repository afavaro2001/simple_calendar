# Simple PHP Calendar
This is a highly customizable calendar developed in PHP 8.2, MaridDb, Bootsrap 5 and FullCalendar (https://fullcalendar.io/docs).

There is a calendar_dashboard.sql MaridDb dump in the root folder with some basic settings.
